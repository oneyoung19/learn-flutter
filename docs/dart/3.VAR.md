`Dart` 是一门强类型语言。

即使它能跟 `JavaScript` 使用 `var` 声明变量。

```dart
void main() {
  // 当使用var声明时，虽然没有显式的说明类型，但Dart会默认推断出类型。
  var a = 1;
  // 类型不能更改
  a = '1';
}
```

```dart
void main() {
  var arr = ['1', '2'];
  // 由于上面的数组只有String类型，因此此处不能加入int类型。
  arr.add(3);
  print(arr);
}
```

简而言之，`Dart` 与 `JavaScript` 的明显区别之一是，`Dart` 具有一套隐式类型系统。

可以看做，**`Dart` 约等于 `JavaScript + TypeScript`**。
